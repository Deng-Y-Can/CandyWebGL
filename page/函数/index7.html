  • Add functions using "+ Add Function".
  • Function Types:
    • Explicit: y = f(x) or z = f(x,y)
    • Implicit: f(x,y) = 0 or f(x,y,z) = 0
    • Parametric: x(t), y(t) or x(u,v), y(u,v), z(u,v)
  • 3D: Use x,y,z; 2D: use x,y.
  • In Parametric: Use comma-separated expressions like "sin(t), cos(t)"
  • In Implicit: Expression should equal 0, e.g., "x^2 + y^2 - 1" for circle
