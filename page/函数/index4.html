Instructions:

  • Add functions using the "+ Add Function" button.
  • For 3D: Use expressions like sin(x)*cos(y). Variables are x and y.
  • For 2D: Use expressions like sin(x). Variable is x. Curve is drawn on the XoY plane.
  • Customize color and visibility for each function.
  • Click "Plot All" to draw all enabled functions.

3D Examples:

  • Paraboloid: x^2 + y^2
  • Saddle: x^2 - y^2
  • Ripple: sin(sqrt(x^2 + y^2)) / (sqrt(x^2 + y^2) + 0.1)

2D Examples:

  • Sine: sin(x)
  • Cosine: cos(x)
  • Parabola: x^2
  • Absolute Value: abs(x)

Functions: sin, cos, tan, asin, acos, atan, log (ln), log10, sqrt, exp, abs.

Operators: +, -, *, /, ^ (power).

Ready. Add a function and click 'Plot All'.
Camera: Pos(0,0,0) Rot(0°,0°,0°)
